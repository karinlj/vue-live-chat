# live-chat

## Description

Live chat app built in Vue 3

## Prerequisites

NodeJS

## Installation

To clone
`git clone https://github.com/karinlj/vue-live-chat.git`

```
npm install
```

## To run

```
npm run serve
```

## To build

```
npm run build
```

Then use the dist folder you get.

## Built With

- vue.js 3
- CSS
